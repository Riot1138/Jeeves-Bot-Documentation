# Playlist Command

This command allows you to view the contents of and control your playlist or audio queue.

### About

**Aliases:** `playlist`  
**Available in DM:** No  
**Can Restrict to a channel:** Yes  
**Global Support:** Yes  

### Examples:

* `!playlist`
  - Shows the current songs in the playlist
* `!playlist clear`
  - Removes the playlist from the server
