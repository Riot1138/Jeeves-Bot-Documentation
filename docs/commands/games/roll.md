# Roll Command
Rolle the dice!

### Details

**Aliases:** `roll`   
**Available in DM:** No   
**Can Restrict to a channel:** Yes    

* The roll command provides a direct imitation of the in-game roll feature.
***
### Examples

* `!roll`  
> Rolls between the number 1 and 100\

* `!roll 53-3023`  
> Rolls between the number 53 and 3023
***
